## 2. 次のフェーズ：プロダクション移行への課題

### Phase 1: インテリジェンス (AI & Search)
- [ ] **LLM Pipeline**: Abstractから4セクション構成を自動生成するプロンプトの実装。
- [ ] **Vector Search**: OpenAI Embeddings を用いた類似論文検索。
- [ ] **Personalization**: 評価ログ（Good/Bad/Dwell Time）に基づくランキングの動的更新。

### Phase 2: 基盤構築 (Infrastructure)
- [ ] **永続化**: PostgreSQL + Prisma への移行（現状は localStorage/Memory）。
- [ ] **Auth**: ユーザーアカウント管理とライブラリの同期。
- [ ] **API Design**: 大量データを扱うためのページネーションとプリフェッチ。

## 3. デザインシステム仕様 (UI Standards)
- **配色**: 背景(Orange), 目的(Purple), 提案(Green), 評価(Blue)。
- **フィードバック**: スワイプ距離 30px で色の予兆、100px で確定。
- **カード形状**: `max-w-2xl`, `h-[70vh]`, `rounded-3xl`（影付き）。
